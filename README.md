# 🌱 EcoTracker – The Butterfly Effect of Everyday Life

EcoTracker is a full-stack web application designed to help individuals understand, monitor, and reduce their carbon footprint. By analyzing daily activities such as energy consumption and travel habits, EcoTracker provides actionable insights and personalized sustainability recommendations that encourage eco-friendly decision-making.

## 🚀 Project Overview

Climate change is one of the biggest challenges facing our world today. Many people want to live more sustainably but don't know how their everyday actions impact the environment.

EcoTracker bridges this gap by providing users with a platform to:

- Measure their carbon footprint
- Track energy consumption
- Analyze travel emissions
- Receive sustainability recommendations
- Monitor progress over time

The project aims to demonstrate how technology can be used to promote environmental awareness and sustainable living.

## ✨ Features

### 🌍 Carbon Footprint Calculator
Calculate carbon emissions generated from daily activities and lifestyle choices.

### ⚡ Energy Consumption Tracking
Monitor appliance usage and estimate the environmental impact of household energy consumption.

### 🚗 Travel Emission Analysis
Analyze emissions generated from different modes of transportation.

### 🤖 AI-Powered Sustainability Suggestions
Receive intelligent recommendations to reduce carbon emissions and adopt greener habits.

### 👤 User Authentication
Secure user registration, login, and profile management.

### 📊 Personal Dashboard
Visualize environmental impact and track sustainability progress.

### 🏆 Carbon Score & Leaderboard
Encourage eco-friendly behavior through scoring and competitive rankings.

### 📈 Progress Monitoring
Track improvements and sustainability achievements over time.

## ⚙️ Installation Guide

### 1. Clone the Repository

```bash
git clone https://github.com/prajwaljadhav9793/EcoTracker.git
```

### 2. Navigate to the Project Directory

```bash
cd EcoTracker
```

### 3. Create a Virtual Environment

```bash
python -m venv venv
```

### 4. Activate the Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the Application

```bash
python app.py
```

### 7. Open in Browser

```text
http://127.0.0.1:5000
```
